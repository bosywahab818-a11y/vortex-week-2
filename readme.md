# Vortex Tech AI & ML Internship - Week 2

## Build a Classification Model

### Project Overview

This project builds a machine learning classification model using the Titanic dataset.

Two models were trained:

- Logistic Regression
- Decision Tree

The models were evaluated using:

- Accuracy
- Precision
- Recall
- F1 Score

### Technologies

- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn
